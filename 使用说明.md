# 宠物管理系统 - 使用说明

## 文件说明

本项目包含以下可执行文件：

### 1. target/宠物管理系统.exe
- **推荐使用**：这是主要的可执行文件
- **要求**：需要 Java 17 或更高版本
- **使用方法**：直接双击运行即可

### 2. target/pet-management-system-1.0.0.jar
- **备用方案**：如果exe文件无法运行，可以使用jar文件
- **使用方法**：
  - 双击运行（如果系统已关联jar文件）
  - 或在命令行中执行：`java -jar pet-management-system-1.0.0.jar`

### 3. 启动宠物管理系统.bat
- **备用启动器**：Windows批处理文件，提供友好的启动体验
- **功能**：自动检测Java环境，提供错误提示
- **使用方法**：直接双击运行

## 系统要求

- **操作系统**：Windows 10 或更高版本
- **Java环境**：Java 17 或更高版本
  - 下载地址：https://adoptium.net/
  - 推荐安装 Eclipse Temurin 17 (LTS)

## 重新打包

如果需要重新打包项目，可以运行以下命令：

```bash
mvn clean package
```

打包完成后，新的文件将生成在 `target` 目录中。

## 故障排除

### 问题1：exe文件无法运行
**解决方案**：
1. 确认已安装Java 17或更高版本
2. 尝试使用 `启动宠物管理系统.bat` 启动，它会提供详细的错误信息
3. 或直接使用jar文件：`java -jar target/pet-management-system-1.0.0.jar`

### 问题2：提示未找到Java
**解决方案**：
1. 下载并安装Java 17：https://adoptium.net/
2. 安装后重新启动应用程序

### 问题3：应用程序启动但界面显示异常
**解决方案**：
1. 确保使用的是Java 17或更高版本
2. 检查系统显卡驱动是否正常

## 数据存储

- 数据库文件自动存储在：`用户主目录/.pet-management-system/pet_db.sqlite`
- 该目录会在首次运行时自动创建

## 技术支持

如有问题，请检查：
1. Java版本是否符合要求
2. 系统是否有足够的权限运行应用程序
3. 防病毒软件是否阻止了应用程序运行
