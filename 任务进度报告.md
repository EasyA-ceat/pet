## 项目架构现状

### 技术架构图
```
┌─────────────────────────────────┐
│         JavaFX UI 界面层         │
├─────────────────────────────────┤
│  控制器层 │ 服务层 │ 工具类库   │
├─────────────────────────────────┤
│       Spring Data JPA 层         │
├─────────────────────────────────┤
│          SQLite 数据库           │
└─────────────────────────────────┘
```

### 核心技术特点

1. **分层架构设计**：清晰的层次划分，职责分明
2. **自动配置**：Spring Boot提供自动配置和快速开发体验
3. **类型安全**：使用Spring Data JPA减少SQL编写
4. **轻量级存储**：SQLite数据库易于部署和维护
5. **跨平台支持**：基于Java和JavaFX，支持Windows等平台
