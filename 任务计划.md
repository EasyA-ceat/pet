## 开发规范
- 遵循 Spring Boot 和 JavaFX 最佳实践
- 使用 4 空格缩进
- 类名使用 PascalCase，方法名和变量名使用 camelCase
- 常量使用 UPPER_SNAKE_CASE
- 每个模块完成后需编写单元测试并进行页面可视化测试
- 数据库操作使用 JPA 和 SQLite
- 图片处理使用 Thumbnailator 库
- 图表生成使用 XChart 库
- PDF 导出使用 iTextPDF 库
